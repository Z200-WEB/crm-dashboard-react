# CRM Operations Dashboard

A modern CRM dashboard built with React and Tailwind CSS to practice real business workflows in a clean, user-friendly UI.

---

## 🇺🇸 English

## Overview
This project focuses on core CRM operations:
- Lead / customer tracking
- Opportunity (deal) management
- Follow-up task management
- KPI visibility through a dashboard

It is an original project for learning and portfolio use.

## Features
- **Lead & Customer Management**
  - Track status: New, Contacted, Qualified, Closed
- **Opportunity Pipeline**
  - Monitor deal stage and estimated value
- **Task Management**
  - Organize follow-up actions
- **Dashboard Metrics**
  - Quick snapshot of key CRM numbers
- **Responsive Design**
  - Works on desktop and tablet

## Tech Stack
- React 18
- Tailwind CSS
- Vite
- JavaScript (ES6+)
